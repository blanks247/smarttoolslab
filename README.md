# ⚡ SmartToolsLab

> Free, fast, and private online tools — no signup, no limits.

**Live site**: [smarttoolslab.com](https://smarttoolslab.com) *(coming soon)*

## 🛠️ Tools Included

| Tool | Description |
|---|---|
| 🖼️ Image Compressor | Compress JPG/PNG/WebP in-browser, no upload |
| 📋 JSON Formatter | Format, validate, and minify JSON instantly |
| 📝 Word Counter | Live word/char count + social media limits |
| 🔲 QR Code Generator | URL/WiFi/Email QR codes, download as PNG |
| 📄 PDF Merge | Client-side PDF merging with pdf-lib |

## 🚀 Tech Stack

- Pure **HTML + CSS + Vanilla JS** — zero build step, zero dependencies
- Client-side only — your files never leave your browser
- CDN libraries: `qrcodejs`, `pdf-lib`

## 💰 Monetization

- Google AdSense placeholder zones on every page
- Affiliate link slots in footer

## 📦 Deploy

This is a **static site** — deploy anywhere for free:

- **Netlify**: Drag & drop the folder, or connect this repo
- **Vercel**: `vercel --prod` or connect via GitHub
- **GitHub Pages**: Enable in repo settings → Pages → main branch
